# trckr.trvlr.ch (Frontend)

## Build setup
```
# Install dependencies
npm install

# Serve with hot reload at localhost:8080
npm run dev

# Build for production with minification
npm run build

# Run unit tests
npm run unit

# Run e2e tests
npm run e2e

# Run all tests
npm test
```
